# Video Generation Skill

This skill uses browser automation to generate short videos using AI tools.

## Process

1. Navigate to the video generation service
2. Enter the prompt or upload source material
3. Configure video settings (duration, style, etc.)
4. Initiate generation
5. Wait for processing to complete
6. Download the result

## Prompt Guidelines

When crafting video prompts:
- Describe the scene or action clearly
- Specify the style (cinematic, animated, etc.)
- Include timing information if relevant
- Mention camera movement if desired
- Describe lighting and atmosphere

## Example Prompts

- "Product showcase video, rotating view of the item, clean white background, 5 seconds"
- "Abstract animated background, flowing gradients, loop-friendly, blue and purple"
- "Text animation for logo reveal, modern and sleek, 3 seconds"

## Browser Automation

Use Claude's Chrome MCP tools to:
1. Navigate to the video generation URL
2. Upload any source images if needed
3. Input the prompt text
4. Configure generation settings
5. Start generation and wait for completion
6. Download the result

## Output

After generation, provide:
- Path to downloaded video
- Description of what was generated
- Video specifications (duration, resolution)
- Any adjustments that might improve results
