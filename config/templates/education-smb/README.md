# Education SMB Team Template

This template provides a complete AI team setup for small and medium-sized education businesses. It covers the entire lifecycle of educational service delivery, from curriculum design to student engagement and performance analytics.

## 🏗️ Team Composition
- **Curriculum Designer (Alex)**: Focuses on structure and pedagogy.
- **Content Creator (Casey)**: Produces engaging lesson materials.
- **Student Engagement Manager (Sam)**: Manages community and feedback.
- **Analytics Specialist (Skyler)**: Provides data-driven insights.

## 🔄 Included Workflows
1. **Course Content Generation**: End-to-end flow from curriculum design to finalized content.
2. **Student Communication**: Automates feedback loops and announcements.
3. **Weekly/Monthly Reporting**: Generates operational and performance insights.

## 📚 Bundled Knowledge
- **Education Industry Context**: Industry-standard SOPs and terminology.
- **Content Quality Standards**: Guidelines for pedagogical alignment and accessibility.

## 🚀 Quick Start
To instantiate this team, use the Crewly CLI:
```bash
npx crewly create team --template education-smb
```

## 🎯 Default Goals
See `goals.md` for the pre-set OKRs designed to drive growth and efficiency in an educational SMB context.
